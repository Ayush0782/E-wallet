#Overview
This project is a Java-based backend system for an E-Wallet application that allows users to securely manage digital money. The system supports user account management, wallet balance handling, and transaction processing.
It is designed following RESTful API principles and demonstrates backend development concepts such as authentication, database management, and secure transaction handling.

#Features
User Registration & Login
Wallet Creation
Add Money to Wallet
Transfer Money Between Users
Transaction History
Balance Inquiry
Secure API endpoints
